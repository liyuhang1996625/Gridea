学习Markdown笔记📝

2019-12-27 3 min read # 笔记notes

我听到Markdown这个词是在2018年夏天一位程序员口中。
感觉很高大上，其实我根本不知道是啥。
不知道是啥咱就学呗，整就完了，下面是一份学习笔记。欢迎指教。

Markdown是什么🤔️

Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。(官方定义)
我认为就是一种写作用符号进行排版的写作方式。

Markdown的怎么用✍️

编写Markdown有很多种语法，常用的有以下几种（其实我只学会了以下几种）

1.标题

在需要设置标题的文字前面加#来形成标题。在#结束的时候添加空格和标题分开，最多支持六级标题。下面是示意图片。（tips：#号结束之后要与标题文字分开📌）

2.字体

字体主要用法有斜体，加粗，斜体加粗，加删除线

  • 加粗
    前后两个**将文字包裹起来
    我是加粗文字
  • 斜体
    前后一个*将文字包裹起来
    我是斜体文字
  • 斜体加粗
    前后三个***将文字包裹起来
    我是斜体加粗文字
  • 删除线
    前后两个将文字包裹起来(tips：输入需要切换到英文输入法)
    我是删除线文字

    字体应该就这几种用法了。继续往下看。

3.引用

引用很简单，在引用的文字前加>就可以了。也可以叠加嵌套引用，类似于盖楼样式。

  • 不加嵌套的引用

我是引用1

我是引用2

我是引用3

  • 添加嵌套引用

我是引用1

我是引用2

我是引用3

4.添加分割线

只需要三个同样的符号或三个以上就可以。

5.图片

图片这个吧，我觉得Markdown的用法并不简单，还不如直接添加图片来的快。

6.无序列表

  • 无序列表
    无序列表用- + *这三个符号任意一个就🉑，注意文字前加空格

    • 我是无序列表1
    • 我是无序列表2
    • 我是无序列表3

  • 有序列表
    数字1 2 3 加点，就🉑️，注意文字前加空格

    1. 我是有序列表1
    2. 我是有序列表2
    3. 我是有序列表3
