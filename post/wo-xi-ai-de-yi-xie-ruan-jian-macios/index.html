我喜爱的一些小软件Mac/iOS

我喜爱的一些小软件Mac/iOS

2020-01-03 3 min read

今天早上，例行在网上学习新知识（冲浪摸鱼）
这款软件解决的我的痛点是我最近发现的好玩的有用的软件越来越多，
导致顶部导航栏越来越长...越来越长...越来越长...（都快到了左侧苹果icon那里了）
机缘巧合发现有人安利一款软件，刚刚好解决了我一个大大的痛点！
我就来汇总一下，我接触到过的并且在一直使用的一些小软件。

第一款

Hidden Bar长下面这样


使用方法

官方给安排的明明白白

简直是拯救我于焦虑之中，我的导航栏里面现在只显示了微信，企业微信，翻译软件，滴答清单，时间日期，电量。（怎么感觉还是好多）我显示这些软件是因为这些软件由于我的个人习惯，大概率是从顶部导航栏触达。这些软件还有消息提示是不能错过的～

第二款

mate translate
它的功能非常单一简单且实用，
就是翻译 translate支持N多语言，（刚才查看了一下有102种语言）自动检测文字语种。
还有词典功能，虽然我一次都没用过，但是也不妨碍我非常喜欢这个软件。

使用方法

在导航栏点击icon，弹出界面直接输入你要翻译的词。

第三款

iPaste
朴实无华的剪贴板工具，相信有这个痛点的人自然会明白，有这么一个剪贴板工具，效率会提升N多倍。
Mac App store可直接搜索

使用方法

官方使用教程

第四款

permute
这小图标很是可爱，不过是收费98元的软件。有句话说的好，没有花钱的不是。

这是一款视频转换格式工具，但是他不仅仅可以转换视频，图片压缩也是可以办到的～（还是极简风）

使用方法

拖进去再选择自己想要的格式就OK啦～

第五款

Foldery
想象力软件，自定义文件夹颜色的软件。长啥样呢👀使用起来so easy，这样就不怕茫茫文件夹当中找不到那一抹绿了～

使用方法

目前这些就是我个人经常使用的就是这几款可以稍微的提高一小下效率的小软件。
